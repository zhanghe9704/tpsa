This is a runtime library for \href{http://www.typescriptlang.org/}{\texttt{ Type\+Script}} that contains all of the Type\+Script helper functions.

This library is primarily used by the {\ttfamily -\/-\/import\+Helpers} flag in Type\+Script. When using {\ttfamily -\/-\/import\+Helpers}, a module that uses helper functions like {\ttfamily \+\_\+\+\_\+extends} and {\ttfamily \+\_\+\+\_\+assign} in the following emitted file\+:
