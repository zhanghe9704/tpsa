CLI arguments parser for node.\+js, with \href{https://docs.python.org/3.9/library/argparse.html\#sub-commands}{\texttt{ sub-\/commands}} support. Port of python\textquotesingle{}s \href{http://docs.python.org/dev/library/argparse.html}{\texttt{ argparse}} (version \href{https://github.com/python/cpython/blob/v3.9.0rc1/Lib/argparse.py}{\texttt{ 3.\+9.\+0}}).

{\bfseries{Difference with original.}}


\begin{DoxyItemize}
\item JS has no keyword arguments support.
\begin{DoxyItemize}
\item Pass options instead\+: `new Argument\+Parser(\{ description\+: \textquotesingle{}example', add\+\_\+help\+: true \}){\ttfamily .}
\end{DoxyItemize}
\item {\ttfamily JS has no python\textquotesingle{}s types}int{\ttfamily ,}float{\ttfamily , ...
\begin{DoxyItemize}
\item Use string-\/typed names\+:.add\+\_\+argument(\textquotesingle{}-\/b\textquotesingle{}, \{ type\+: \textquotesingle{}int\textquotesingle{}, help\+: \textquotesingle{}help\textquotesingle{} \}){\ttfamily . -\/}r{\ttfamily format specifier uses}require(\textquotesingle{}util\textquotesingle{}).inspect()\`{}.
\end{DoxyItemize}}
\end{DoxyItemize}
