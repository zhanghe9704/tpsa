That\textquotesingle{}s what {\ttfamily at-\/least-\/node} is for.

When passing in a version string\+:


\begin{DoxyItemize}
\item You cannot include a leading {\ttfamily v} (i.\+e. {\ttfamily v10.\+12.\+0})
\item You cannot omit sections (i.\+e. {\ttfamily 10.\+12})
\item You cannot use pre-\/releases (i.\+e. {\ttfamily 1.\+0.\+0-\/beta})
\item There is no input validation, if you make a mistake, the resulting behavior is undefined 
\end{DoxyItemize}
