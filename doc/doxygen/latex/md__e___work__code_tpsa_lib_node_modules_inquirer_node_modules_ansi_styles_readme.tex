\hypertarget{md__e___work__code_tpsa_lib_node_modules_inquirer_node_modules_ansi_styles_readme_autotoc_md1374}{}\doxysection{$<$a href=\char`\"{}https\+://gist.\+github.\+com/\+XVilka/8346728\char`\"{}$>$256 / 16 million (\+True\+Color) support$<$/a$>$}\label{md__e___work__code_tpsa_lib_node_modules_inquirer_node_modules_ansi_styles_readme_autotoc_md1374}
{\ttfamily ansi-\/styles} uses the \href{https://github.com/Qix-/color-convert}{\texttt{ {\ttfamily color-\/convert}}} package to allow for converting between various colors and ANSI escapes, with support for 256 and 16 million colors.

The following color spaces from {\ttfamily color-\/convert} are supported\+:


\begin{DoxyItemize}
\item {\ttfamily rgb}
\item {\ttfamily hex}
\item {\ttfamily keyword}
\item {\ttfamily hsl}
\item {\ttfamily hsv}
\item {\ttfamily hwb}
\item {\ttfamily ansi}
\item {\ttfamily ansi256}
\end{DoxyItemize}

To use these, call the associated conversion function with the intended output, for example\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{style.color.ansi.rgb(100, 200, 15); // RGB to 16 color ansi foreground code}
\DoxyCodeLine{style.bgColor.ansi.rgb(100, 200, 15); // RGB to 16 color ansi background code}
\DoxyCodeLine{}
\DoxyCodeLine{style.color.ansi256.hsl(120, 100, 60); // HSL to 256 color ansi foreground code}
\DoxyCodeLine{style.bgColor.ansi256.hsl(120, 100, 60); // HSL to 256 color ansi foreground code}
\DoxyCodeLine{}
\DoxyCodeLine{style.color.ansi16m.hex('\#C0FFEE'); // Hex (RGB) to 16 million color foreground code}
\DoxyCodeLine{style.bgColor.ansi16m.hex('\#C0FFEE'); // Hex (RGB) to 16 million color background code}

\end{DoxyCode}
