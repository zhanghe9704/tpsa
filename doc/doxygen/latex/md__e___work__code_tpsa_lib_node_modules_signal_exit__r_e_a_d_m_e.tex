When you want to fire an event no matter how a process exits\+:


\begin{DoxyItemize}
\item reaching the end of execution.
\item explicitly having {\ttfamily process.\+exit(code)} called.
\item having {\ttfamily process.\+kill(pid, sig)} called.
\item receiving a fatal signal from outside the process
\end{DoxyItemize}

Use {\ttfamily signal-\/exit}.

The return value of the function is a function that will remove the handler.

Note that the function {\itshape only} fires for signals if the signal would cause the process to exit. That is, there are no other listeners, and it is a fatal signal.\hypertarget{md__e___work__code_tpsa_lib_node_modules_signal_exit__r_e_a_d_m_e_autotoc_md2899}{}\doxysection{Options}\label{md__e___work__code_tpsa_lib_node_modules_signal_exit__r_e_a_d_m_e_autotoc_md2899}
