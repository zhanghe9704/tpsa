Strip comments from JSON. Lets you use comments in your JSON files! \end{quote}
This is now possible\+:

It will replace single-\/line comments {\ttfamily //} and multi-\/line comments {\ttfamily /$\ast$$\ast$/} with whitespace. This allows JSON error positions to remain as close as possible to the original source.
