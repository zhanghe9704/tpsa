Determine columns needed for a fixed-\/size wide-\/character string

\DoxyHorRuler{0}


wcwidth is a simple Java\+Script port of \href{http://man7.org/linux/man-pages/man3/wcswidth.3.html}{\texttt{ wcwidth}} implemented in C by Markus Kuhn.

Java\+Script port \href{https://github.com/mycoboco/wcwidth.js}{\texttt{ originally}} written by Woong Jun \href{mailto:woong.jun@gmail.com}{\texttt{ woong.\+jun@gmail.\+com}} (\href{http://code.woong.org/}{\texttt{ http\+://code.\+woong.\+org/}})\hypertarget{md__e___work__code_tpsa_lib_node_modules_wcwidth__readme_autotoc_md3191}{}\doxysection{Example}\label{md__e___work__code_tpsa_lib_node_modules_wcwidth__readme_autotoc_md3191}

{\ttfamily wcwidth()} and its string version, {\ttfamily wcswidth()} are defined by IEEE Std 1002.\+1-\/2001, a.\+k.\+a. POSIX.\+1-\/2001, and return the number of columns used to represent the given wide character and string.

Markus\textquotesingle{}s implementation assumes the wide character given to those functions to be encoded in ISO 10646, which is almost true for Java\+Script\textquotesingle{}s characters.
