Returns an array of callsite objects with the following methods\+:


\begin{DoxyItemize}
\item {\ttfamily get\+This}\+: returns the value of {\ttfamily this}.
\item {\ttfamily get\+Type\+Name}\+: returns the type of {\ttfamily this} as a string. This is the name of the function stored in the constructor field of {\ttfamily this}, if available, otherwise the object\textquotesingle{}s {\ttfamily \mbox{[}\mbox{[}Class\mbox{]}\mbox{]}} internal property.
\item {\ttfamily get\+Function}\+: returns the current function.
\item {\ttfamily get\+Function\+Name}\+: returns the name of the current function, typically its {\ttfamily name} property. If a name property is not available an attempt will be made to try to infer a name from the function\textquotesingle{}s context.
\item {\ttfamily get\+Method\+Name}\+: returns the name of the property of {\ttfamily this} or one of its prototypes that holds the current function.
\item {\ttfamily get\+File\+Name}\+: if this function was defined in a script returns the name of the script.
\item {\ttfamily get\+Line\+Number}\+: if this function was defined in a script returns the current line number.
\item {\ttfamily get\+Column\+Number}\+: if this function was defined in a script returns the current column number
\item {\ttfamily get\+Eval\+Origin}\+: if this function was created using a call to {\ttfamily eval} returns a string representing the location where {\ttfamily eval} was called.
\item {\ttfamily is\+Toplevel}\+: is this a top-\/level invocation, that is, is this the global object?
\item {\ttfamily is\+Eval}\+: does this call take place in code defined by a call to {\ttfamily eval}?
\item {\ttfamily is\+Native}\+: is this call in native V8 code?
\item {\ttfamily is\+Constructor}\+: is this a constructor call?
