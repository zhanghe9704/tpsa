In Node.\+js, this module simply re-\/exports the {\ttfamily util.\+deprecate()} function.

In the web browser (i.\+e. via browserify), a browser-\/specific implementation of the {\ttfamily util.\+deprecate()} function is used.\hypertarget{md__e___work__code_tpsa_lib_node_modules_util_deprecate__r_e_a_d_m_e_autotoc_md3166}{}\doxysection{API}\label{md__e___work__code_tpsa_lib_node_modules_util_deprecate__r_e_a_d_m_e_autotoc_md3166}
A {\ttfamily deprecate()} function is the only thing exposed by this module.
