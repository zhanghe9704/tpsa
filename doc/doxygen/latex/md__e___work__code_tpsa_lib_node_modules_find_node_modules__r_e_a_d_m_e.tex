This is a little node module to find the path of every parent node\+\_\+modules directory. It\textquotesingle{}s useful for things like Sass, where you can\textquotesingle{}t specify the exact path to individual modules (in which case \href{https://www.npmjs.com/package/findup-sync}{\texttt{ findup-\/sync}} would be sufficient), and you can\textquotesingle{}t just give an array of parent node\+\_\+modules which might exist, because it will error if they don\textquotesingle{}t.

In most cases you\textquotesingle{}re trying to find node\+\_\+modules directories, findup-\/sync should be sufficient. This library is specifically for if you want an array containing all the parent node\+\_\+modules paths.
