Find the first file matching a given pattern in the current directory or the nearest ancestor directory.

Matching is done with \href{http://github.com/jonschlinkert/micromatch}{\texttt{ micromatch}}, please report any matching related issues on that repository.\hypertarget{md__e___work__code_tpsa_lib_node_modules_findup_sync__r_e_a_d_m_e_autotoc_md1077}{}\doxysubsection{Usage}\label{md__e___work__code_tpsa_lib_node_modules_findup_sync__r_e_a_d_m_e_autotoc_md1077}
