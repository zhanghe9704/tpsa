\hypertarget{md__e___work__code_tpsa_lib_node_modules_concat_map__r_e_a_d_m_e_autotoc_md861}{}\doxysubsection{concat\+Map(xs, fn)}\label{md__e___work__code_tpsa_lib_node_modules_concat_map__r_e_a_d_m_e_autotoc_md861}
Return an array of concatenated elements by calling {\ttfamily fn(x, i)} for each element {\ttfamily x} and each index {\ttfamily i} in the array {\ttfamily xs}.

When {\ttfamily fn(x, i)} returns an array, its result will be concatenated with the result array.
