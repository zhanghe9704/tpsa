\hypertarget{md__e___work__code_tpsa_lib_node_modules_homedir_polyfill__r_e_a_d_m_e_autotoc_md1289}{}\doxysection{Reasoning}\label{md__e___work__code_tpsa_lib_node_modules_homedir_polyfill__r_e_a_d_m_e_autotoc_md1289}
This library is a polyfill for the \href{https://nodejs.org/api/os.html\#os_os_homedir}{\texttt{ node.\+js os.\+homedir}} method found in modern versions of node.\+js.

This implementation tries to follow the implementation found in {\ttfamily libuv} by finding the current user using the {\ttfamily process.\+geteuid()} method and the {\ttfamily /etc/passwd} file. This should usually work in a linux environment, but will also fallback to looking at user specific environment variables to build the user\textquotesingle{}s home directory if neccessary.

Since {\ttfamily /etc/passwd} is not available on windows platforms, this implementation will use environment variables to find the home directory.

In modern versions of node.\+js, \href{https://nodejs.org/api/os.html\#os_os_homedir}{\texttt{ os.\+homedir}} is used.\hypertarget{md__e___work__code_tpsa_lib_node_modules_homedir_polyfill__r_e_a_d_m_e_autotoc_md1290}{}\doxysection{About}\label{md__e___work__code_tpsa_lib_node_modules_homedir_polyfill__r_e_a_d_m_e_autotoc_md1290}
