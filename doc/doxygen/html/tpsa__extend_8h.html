Composition of two groups of TPS vectors. Calculate the composition of two groups of TPS vectors, and save the result into the third group of TPS vectors. Call the first group of TPS vectors, ivecs, as f, which includes n TPS vectors: f_1, f_2, ..., f_n. The second group of TPS vectors, v, should contain the same number of TPS vectors as the number of TPS bases. Call the third group of TPS vectors, ovecs, as g, which also includes n TPS vectors: g_1, g_2, ..., g_n. g = f(v), or g_1 = f_1(v), g_2 = f_2(v), ..., g_n = f_n(v). This is an alternative function for the original one (ad_subst) in tpsa.cpp.

Parameters
[in]ivecsA group of TPS vectors, saved as an std::vector.
